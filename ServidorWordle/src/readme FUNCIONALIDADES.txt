Descripción de las Funcionalidades
	Autenticación del Usuario: Se autentica al usuario con las credenciales proporcionadas.
	Inicio del Juego: Se inicia una nueva partida para el usuario autenticado.
	Envío de Preguntas: Se selecciona una pregunta al azar de la base de datos y se envía al cliente.
	Procesamiento de Respuestas: Se recibe la respuesta del cliente, se verifica si es correcta y se actualiza la puntuación en consecuencia.
	Guardar Estado del Juego: Se guarda el estado actual del juego cuando el usuario decide retirarse.
	Finalización del Juego: Se finaliza el juego y se califican los aciertos cuando el usuario decide finalizar el juego.
